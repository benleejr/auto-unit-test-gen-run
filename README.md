FLASK SERVER SETUP
Create a virtual environment within your project folder and install
 Flask there. This is considered best practice for Python development.

 To host Flask on your local machine:
 python3 -m venv venv
 source venv/bin/activate  
 
 On Windows: venv\Scripts\activate
 
 pip install Flask
 pip install coverage
'Flask run' to run a local Flask server.
